School Project:
Beach planning app

Wave2Me was my main project at CareerFoundry. It is a beach planning app that helps users plan their day at the beach. It provides suggestions and social media so users can decide the best activity for their skill level.


Freelance Project:
Game store website

This website is a Freelance project I created for a small card and board gamestore in Garland Texas. My client requested a simple site similar to what their competitors have. The site features a web application to help plan Role Playing Game's.
